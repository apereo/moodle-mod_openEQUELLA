Moodle module for openEQUELLA integration

Requirements

  • This module requires openEQUELLA 2018.2 or higher.
  • The master branch works with the Moodle 3.5 and up.

For Moodle earlier versions, choose the appropriate git branches.

Support

Feel free to fork this repository and submit Pull Requests for any defects or features you'd like to address or contribute. Opening issues here is also recommended.

If you'd like to discuss an issue or some code, then come and chat with us on one of the following platforms:

  • Users mailing list - Most active, best for more usage centric questions
  • Developer mailing list - For discussing work on the code
